# Blobby Volley: LÖVE Edition

A port of [Blobby Volley 2](https://github.com/danielknobe/blobbyvolley2) using the *awesome* Lua game framework [LÖVE](https://love2d.org/).

## Requirements
- LÖVE 11.3 or higher
